# Runtime-only release package

Це зменшений пакет для запуску.

## Що включено
- `index.html`
- локальні `css/`, які підключені в `index.html`
- локальні `js/`, які підключені в `index.html`
- `img/`

## Що навмисно НЕ включено
- `PHASE_*.md`
- `FINAL_RELEASE_UA.md`
- `FILE_MAP_UA.md`
- `_combo_check.js`
- файли, не підключені в `index.html`

## Примітка
`i18n.js` не змінювався в цьому етапі.
