Налаштування / Імпорт

Імпорт таблиці та налаштування мапінгу колонок

✕

1. Джерело імпорту

Завантаж файл або встав публічне CSV-посилання.

Поточне джерело
Файл ще не завантажено.
Статус
Завантаж файл або посилання, визнач колонки, застосуй імпорт.

2. Обов'язкові колонки

Ім’я гравця, тип військ, тір, розмір маршу, розмір групової атаки, альянс, готовність бути капітаном, доступність по зміні.

3. Додаткові колонки

Рівень лігва, резервний тип військ: боєць / наїздник / стрілець та твої власні додаткові колонки.

4. Видимі колонки

Обери, які додаткові колонки треба показувати в таблиці гравців при натисканні «Показати всі дані».

5. Скидання даних

Обережно: ці дії скидають збережені налаштування або дані таблиць.

Підтримуються CSV/XLSX та публічне CSV-посилання Google Sheets.

Фінальний план

Вигляд для офіцерів / скріншота / PDF. Експорт PNG/PDF працює через JS.
